Deploy Amazon Prime Clone Application AWS using DevSecOps Approach

Today I turned a simple idea into a real-time deployment by building and deploying a Prime Video–style application on Kubernetes. What started as a learning exercise became a deep dive into containers, services, networking, and real-world troubleshooting. I containerized the application using Docker, deployed it on a K3s cluster, configured services and ingress for external access, and handled pod-level debugging when things didn’t go as expected. Every error message taught me something new — from port mappings to service communication inside the cluster. This project wasn’t just about deployment; it was about understanding how modern streaming platforms can be architected using cloud-native technologies. Step by step, command by command, I strengthened my DevOps skills and gained more confidence in working with Kubernetes in production-like environments. Learning by building is always powerful — and this is just the beginning
